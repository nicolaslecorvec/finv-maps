# finv-maps

Static cartographic files used on [finv.earth](https://finv.earth/).

This repository stores lightweight map assets, vector tiles and related files used to publish interactive cartographic visualisations on the fiñv website.

## Contents

### `rga_2020_2026_communes_arrondissements.pmtiles`

PMTiles vector tile archive used for the interactive fiñv map comparing the French clay shrink-swell exposure classification between the 2020 and 2026 RGA maps, at commune and municipal arrondissement scale.

Related article:

* [2026 clay shrink-swell map: geology has not changed, risk has](https://finv.earth/en/knowledge/shrink-swell-map-2026-geology-risk/)
* [Carte RGA 2026 : la géologie n’a pas changé, le risque oui](https://finv.earth/connaissance/carte-rga-2026-geologie-risque/)

## Data sources

This derived visualisation is based on public data from:

* BRGM
* Géorisques
* IGN ADMIN EXPRESS

The map is intended for territorial-scale visualisation and comparison. It does not replace the official regulatory RGA exposure map, local geotechnical expertise or parcel-scale soil studies.
