# finv-maps

Static cartographic files used on [finv.earth](https://finv.earth).

## Contents

- `rga_2020_2026_communes_arrondissements.pmtiles`  
  Vector tiles used for the interactive fiñv map comparing RGA exposure classifications between 2020 and 2026 by commune and municipal arrondissement.
